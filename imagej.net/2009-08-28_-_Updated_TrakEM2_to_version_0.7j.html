2009-08-28 - Updated TrakEM2 to version 0.7j

TrakEM2 has been updated to version 0.7j

Please call "Help - Update Fiji" to get the new TrakEM2_.jar into your plugins folder.

The update includes:

  • New AreaList surface and maximum diameter measurements. See arealist measurements.
  • Dialog for importing stacks appears only when it's not obvious which stack to import.
  • New paint mode for AreaList: overlap (current), exclusion and erosion.
  • Improved lens distortion correction.

See all news here.


Thanks to Javier Cabrera Chaves, Christian Niedworok and Stephan Saalfeld for testing, bug reports, features and fixes.
