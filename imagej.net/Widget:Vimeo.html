Widget:Vimeo

This widget allows you to add a Vimeo video player to your wiki page.

Using this widget

Supported parameters include:

  • id - ID of the video to embed
  • caption - description with link to video (default is "[view on Vimeo]")
  • width - width of the video (default is 425)
  • height - height of the video (default is 355)

Copy to your site

To use this widget on your site, just install the MediaWiki Widgets extension and the copy full source code of this page to your wiki, as an article called Widget:Vimeo.
