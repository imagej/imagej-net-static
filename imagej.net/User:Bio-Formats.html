User:Bio-Formats

This account is used to manage the Bio-Formats update site. See also list of update sites.

To activate it in your Fiji setup:

  • Help › Update...
  • Click Manage update sites
  • Check Bio-Formats
  • Click Close

Some files should appear on the list with Status/Action of Update it.

  • Click Apply changes
  • Restart Fiji
  • Help › About Plugins › Bio-Formats Plugins...

The dialog should now show a very recent build date (within the last few days).
