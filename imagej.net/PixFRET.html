PixFRET (Fiji)
Author Jerome Feige, Daniel Sage, Walter Wahli, Beatrice Desvergne, Laurent Gelman
Maintainer Jan Eglinger
Source on GitHub
Initial release 2005
Latest version 1.5.2
Development status stable

Installation

To install the plugin, activate the PixFRET update site and restart Fiji.

Please refer to the user guide (PDF).
