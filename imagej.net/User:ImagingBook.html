User:ImagingBook

Wilhelm Burger
GitHub imagingbook

This account is used to manage the ImagingBook update site. See also list of update sites.

To activate it:

  • Help › Update...
  • Click Manage update sites
  • Check one of the associated update sites:
    • ImagingBook
    • ImagingBookEn1
    • ImagingBookEn2
    • ImagingBookDe2
    • ImagingBookDe3
  • Click Close

Some files should appear on the list with Status/Action of Install it.

  • Click Apply changes
  • Restart ImageJ
