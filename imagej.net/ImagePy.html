ImagePy

ImagePy (ImagePy)
Author Yan Xiaolong
Maintainer Yan Xiaolong
Source on GitHub
Development status active
Website https://www.imagepy.org/

ImagePy is a Python-based extensible image processing framework, inspired by and modeled after the ImageJ user experience. It is built on libraries of the PyData software stack, including NumPy, SciPy, scikit-image, pandas and others.

Publication

  • Wang, A.; Yan, X. & Wei, Z. (2018), "ImagePy: an open-source, Python-based and platform-independent software package for bioimage analysis", Bioinformatics 34(18): 3238-3240, PMID 29718132, doi:10.1093/bioinformatics/bty313, <https://academic.oup.com/bioinformatics/article/34/18/3238/4989871> (on Google Scholar).
