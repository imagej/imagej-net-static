OMERO

OMERO (OMERO)
Author Open Microscopy Environment
Maintainer Open Microscopy Environment
Source on GitHub
Development status active
Website http://www.openmicroscopy.org/info/omero

OMERO is client-server software for visualization, management and analysis of biological microscope images.

Publication

  • Allan, C., Burel, J. M., Moore, J., Blackburn, C., Linkert, M., Loynton, S., ... & Swedlow, J. R. (2012). OMERO: flexible, model-driven data management for experimental biology. Nature methods, 9(3), 245-253.

See also

  • The ImageJ-OMERO project for bidirectional interoperability between ImageJ and OMERO.
