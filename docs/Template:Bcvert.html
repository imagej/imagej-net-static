Template:Bcvert

This template provides a BreadCrumb trail through menu paths and similar hierarchical structures.

Usage

For example, typing: 

The {{key|L}} key is the shortcut for {{bcvert|Plugins|Utilities|Find Commands}}.

will be rendered as:

The L key is the shortcut for
Plugins
▼
Utilities
▼
Find Commands

Up to 10 menu elements can be used: 

{{bcvert|Menu1|Menu2|Menu3|(...)|Menu9|Command}}

will be rendered as:

Menu1
▼
Menu2
▼
Menu3
▼
(...)
▼
Menu9
▼
Command

If the parameter color is set to white, arrows will be displayed white otherwise the default arrow is displayed filled with black for better visibility. 

{{bcvert|color=white|Menu1|Menu2|Menu3|(...)|Menu9|Command}}

will be rendered as:

Menu1
▽
Menu2
▽
Menu3
▽
(...)
▽
Menu9
▽
Command

If the parameter float is set to right, the table will float right, as though align: right was set: 

{{bcvert|float=right|Menu1|Menu2|Menu3|(...)|Menu9|Command}}

will be rendered (inline) as:

Menu1
▼
Menu2
▼
Menu3
▼
(...)
▼
Menu9
▼
Command
