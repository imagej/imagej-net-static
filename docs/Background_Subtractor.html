Background Subtractor

Background Subtractor (ImageJ)
Author MOSAIC Group, Center for Systems Biology Dresden (CSBD), Max Planck Institute of Molecular Cell Biology and Genetics (MPI-CBG), Dresden, Germany.
Update site MOSAIC ToolSuite
Maintainer Krzysztof.Gonciarz
Source MPI-CBG git
Initial release 2008
Development status active
Category Filtering
Website MosaicSuite


Background Subtractor is a tool for removing uneven background from fluorescence microscopy images. It implements a robust histogram-based rolling-ball algorithm and is part of the MosaicSuite, which also offers 3D particle tracking, image segmentation, interaction analysis, and much more. The best way to install it is via the MOSAIC Fiji Update site, as described on the MOSAIC web page.
