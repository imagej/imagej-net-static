3D Viewer

3D Viewer: hardware-accelerated 3D visualization of image stacks as volumes, orthoslices and meshes.

Supports 3D text and scale bars.

The above meshes were generated in TrakEM2, which seamlessly interacts with the 3D Viewer. They represent secondary neuronal lineages from a Drosophila 3rd instar brain.
