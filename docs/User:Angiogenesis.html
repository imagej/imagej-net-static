User:Angiogenesis

This account is used to manage the Angiogenesis update site. See also list of update sites.

To activate it:

  • Help › Update...
  • Click Manage update sites
  • Check Angiogenesis
  • Click Close

Some files should appear on the list with Status/Action of Update it.

  • Click Apply changes
  • Restart ImageJ

Your ImageJ should now possess additional commands!
